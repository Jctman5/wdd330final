# 🎸 GuitarKit

## Your All-In-One Guitar Learning Toolkit

GuitarKit is a responsive web application designed to help guitar players learn chords, explore scales, practice songs, and tune their instrument.

The application combines external APIs, JSON data, local storage, and browser-based audio features to create an interactive guitar reference tool.

---

# Features

## 🎵 Chord Lookup

Users can search guitar chords and view:

- Chord name
- Chord type
- Difficulty level
- Fret positions
- Finger placements
- Notes included
- Chord variations


## 🎼 Song Search

Users can search songs and retrieve:

- Artist information
- Song lyrics
- Practice material


## 🎸 Guitar Tuner

The built-in tuner uses the Web Audio API.

Supported tunings:

- Standard EADGBE
- Drop D
- Open G
- DADGAD


## ⭐ Favorites

Users can save:

- Favorite chords
- Practice songs

Data persists using browser LocalStorage.


## 📚 Search History

Recently viewed chords are stored locally for quick access.


## 🎹 Chord Progression Builder

Users can:

- Create custom progressions
- Save progressions
- Practice transitions


## 🎯 Daily Challenge

A daily chord challenge encourages users to practice unfamiliar shapes.
